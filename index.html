1. Общая информация

Настоящая Политика конфиденциальности описывает, как игра «Легенды Клика RPG» обрабатывает данные пользователей. Игра является мобильным игровым приложением для Android в жанре idle/clicker RPG.

2. Какие данные обрабатывает приложение

Игра может сохранять игровой прогресс на устройстве пользователя, включая: количество золота, кристаллов, текущий уровень, улучшения, достижения, настройки звука, прогресс заданий и другие игровые параметры.

Эти данные используются только для работы игрового процесса и сохранения прогресса.

3. Персональные данные

Приложение не требует регистрации, не запрашивает имя, номер телефона, адрес проживания, паспортные данные или другие подобные персональные сведения.

4. Реклама и сторонние сервисы

В приложении могут использоваться рекламные сервисы и SDK для показа рекламы, получения наград за просмотр рекламы, а также для обеспечения корректной работы рекламных функций.

Такие сторонние сервисы могут обрабатывать технические данные устройства, рекламный идентификатор, сведения о показах рекламы и взаимодействии с рекламой в соответствии со своими правилами конфиденциальности.

5. Покупки и внутриигровые награды

Игра может содержать внутриигровую валюту, награды, задания, достижения, сундуки, события и другие игровые механики. Информация о таких действиях используется для начисления наград и сохранения прогресса.

6. Хранение данных

Основной игровой прогресс хранится локально на устройстве пользователя. При удалении приложения часть локальных данных может быть удалена вместе с приложением.

7. Передача данных третьим лицам

Разработчик не продаёт персональные данные пользователей. Передача технических данных возможна только в рамках работы подключённых сторонних сервисов, например рекламных SDK, если они используются в опубликованной версии приложения.

8. Дети

Приложение не предназначено для намеренного сбора персональных данных детей. Если вы считаете, что ребёнок передал персональные данные через приложение, свяжитесь с разработчиком по указанной почте.

9. Безопасность

Разработчик принимает разумные меры для защиты данных, используемых приложением. Однако ни один способ хранения или передачи данных не может быть гарантирован как абсолютно безопасный.

10. Изменения политики

Разработчик может обновлять настоящую Политику конфиденциальности. Новая версия будет опубликована на этой странице.
