1. Общие положения

Настоящая Политика конфиденциальности описывает, какие данные могут обрабатываться при использовании мобильной игры «Легенды Клика RPG», далее — «Игра» или «Приложение».

Разработчиком и оператором Приложения является Саравас Николай Эдуардович. Используя Игру, пользователь подтверждает, что ознакомился с данной Политикой. Если пользователь не согласен с условиями Политики, он может прекратить использование Игры.

Игра не требует регистрации, создания аккаунта, ввода номера телефона, адреса, паспортных данных или платёжной информации.

2. Какие данные могут обрабатываться

Приложение может обрабатывать ограниченный набор данных, необходимых для работы игры, сохранения локального прогресса и показа добровольной rewarded-рекламы.

2.1. ID устройства или другие идентификаторы

В приложении используется рекламный SDK VK Ads / myTarget. Для работы рекламы рекламная сеть может обрабатывать рекламный идентификатор устройства и другие технические идентификаторы.

Эти данные могут использоваться для показа рекламы, ограничения повторяющихся рекламных показов, проверки корректной работы рекламы, диагностики рекламного SDK и предотвращения злоупотреблений.

2.2. Взаимодействие с приложением

Приложение и рекламный SDK могут обрабатывать данные о взаимодействии с отдельными функциями приложения, например:

  • факт запуска приложения;
  • взаимодействие с рекламным объявлением;
  • просмотр rewarded-рекламы;
  • получение внутриигрового бонуса после добровольного просмотра рекламы.

2.3. Диагностика

Для стабильной работы приложения и рекламного SDK могут обрабатываться технические диагностические данные, связанные с работой приложения, загрузкой рекламы и совместимостью устройства.

2.4. Отчёты об ошибках

Приложение или используемые SDK могут обрабатывать технические данные об ошибках, сбоях и проблемах работы приложения или рекламы. Эти данные используются для диагностики, исправления ошибок и повышения стабильности приложения.

3. Локальные игровые данные

Основной игровой прогресс хранится локально на устройстве пользователя. К таким данным относятся:

  • текущий уровень, открытый максимальный уровень, мир, глава и волна;
  • количество игровой валюты: золото и кристаллы;
  • очки умений и прокачка навыков героя;
  • улучшения героя, помощников, ферм и игровых построек;
  • характеристики героя: урон клика, авто-урон, здоровье, броня и другие игровые параметры;
  • прогресс ежедневных, еженедельных и других игровых заданий;
  • прогресс достижений и информация о получении наград;
  • настройки звука, музыки, вибрации и интерфейса;
  • выбранные аватары, рамки, предметы, артефакты и элементы коллекции.

Эти данные используются для сохранения прогресса между запусками Игры. При удалении Игры с устройства, очистке данных приложения или сбросе устройства локальные данные могут быть удалены.

Если в игре используется имя героя или никнейм, оно применяется только внутри игры для отображения в интерфейсе и хранится локально на устройстве. Разработчик не получает эти данные и не передаёт их третьим лицам.

4. Реклама VK Ads / myTarget

В Игре используется добровольная rewarded-реклама VK Ads / myTarget.

Реклама не является обязательной для прохождения игры. Пользователь сам выбирает, смотреть ли рекламное видео для получения дополнительного внутриигрового бонуса. Если пользователь не хочет смотреть рекламу, он может продолжать играть без просмотра рекламных объявлений.

Примеры добровольных игровых бонусов:

  • временное усиление героя;
  • дополнительная игровая награда;
  • ускорение прогресса;
  • бонус из офлайн-сундука.

Рекламная сеть может обрабатывать технические данные, необходимые для загрузки, показа и диагностики рекламных объявлений, а также для предотвращения мошенничества и злоупотреблений.

5. Передача данных третьим лицам

Разработчик не продаёт персональные данные пользователей.

Для работы добровольной rewarded-рекламы часть технических данных может передаваться рекламной сети VK Ads / myTarget. Такая передача нужна только для работы рекламы, начисления внутриигровых бонусов после просмотра рекламы, диагностики и предотвращения злоупотреблений.

Сторонние сервисы могут обрабатывать данные в соответствии со своими собственными политиками конфиденциальности и условиями использования.

6. Какие данные приложение не запрашивает

Приложение не требует и не запрашивает:

  • номер телефона;
  • адрес электронной почты для регистрации;
  • паспортные данные;
  • домашний адрес;
  • банковские карты;
  • платёжные данные;
  • список контактов;
  • фотографии и видео пользователя;
  • аудиозаписи пользователя;
  • сообщения пользователя;
  • точную геолокацию;
  • данные о здоровье;
  • данные календаря.

7. Разрешения приложения

Игра может использовать следующие разрешения Android:

  • INTERNET — для загрузки рекламных объявлений и работы сетевых функций SDK;
  • ACCESS_NETWORK_STATE — для проверки доступности сети;
  • AD_ID — для корректной работы рекламного идентификатора устройства при использовании рекламы.

Игра не запрашивает доступ к контактам, SMS, звонкам, камере, микрофону, точной геолокации, календарю или пользовательским файлам без отдельного разрешения пользователя.

8. Цели обработки данных

Данные могут использоваться для следующих целей:

  • обеспечение работы приложения;
  • сохранение игрового прогресса на устройстве;
  • работа боевой системы, уровней, миров, врагов и боссов;
  • работа магазина, улучшений, умений героя, помощников, заданий и достижений;
  • отображение количества золота, кристаллов и других игровых показателей;
  • сохранение настроек звука, музыки, вибрации и интерфейса;
  • показ добровольной rewarded-рекламы;
  • выдача внутриигровых бонусов после добровольного просмотра рекламы;
  • диагностика ошибок;
  • повышение стабильности приложения;
  • предотвращение мошенничества и злоупотреблений;
  • соблюдение требований рекламных сетей и магазинов приложений.

9. Возрастная категория

Приложение предназначено для пользователей возрастной категории 12+.

Игра содержит фэнтези-сражения с монстрами и боссами, но не содержит реалистичной жестокости, крови, сцен насилия над людьми или откровенного контента.

Если пользователь является несовершеннолетним, он должен использовать Игру с разрешения родителя или законного представителя.

10. Срок хранения данных

Локальные игровые данные хранятся на устройстве пользователя до удаления приложения, очистки данных приложения или сброса прогресса.

Технические данные, которые могут обрабатываться рекламным SDK, хранятся в соответствии с правилами соответствующей рекламной сети.

11. Безопасность данных

Разработчик принимает разумные меры для защиты данных пользователя. Основной игровой прогресс хранится локально на устройстве пользователя.

Данные, которые могут обрабатываться рекламным SDK, передаются и обрабатываются в соответствии с правилами и техническими механизмами соответствующей рекламной сети.

Несмотря на это, ни один способ передачи данных через Интернет и ни один способ электронного хранения не может быть гарантированно защищён на 100%.

12. Права пользователя

Пользователь может:

  • удалить приложение с устройства;
  • очистить данные приложения в настройках Android;
  • отказаться от просмотра добровольной рекламы;
  • ограничить использование рекламного идентификатора в настройках устройства, если такая возможность доступна;
  • обратиться к разработчику по вопросам обработки данных.

14. Отказ от рекламы и ограничение рекламного идентификатора

Пользователь может не смотреть добровольную rewarded-рекламу. Просмотр рекламы не является обязательным для прохождения игры.

Пользователь может ограничить персонализацию рекламы через настройки своего устройства или аккаунта Google, если такая возможность доступна на устройстве.

Также пользователь может отключить или сбросить рекламный идентификатор устройства в настройках Android, если такая функция поддерживается устройством.
